# 🔴 Red Portal

A static web portal featuring unblocked games, an HTML executor, a website fetcher, and more.

## Features

- 🎮 **Games** — 25+ browser games (Cookie Clicker, Eaglercraft, Slope, and more)
- 💻 **HTML Executor** — Run custom HTML/JS/CSS in a new tab, or upload an HTML file
- 🌐 **Website Fetcher** — Fetch and open any URL via blob URL
- 📋 **Google Form** — Embedded feedback form
- 🏆 **Credits** — Project credits

## Deploy to Vercel

Click the button below to instantly deploy your own copy:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/VVVULTURE/Red-Portal)

Or connect the repo manually:

1. Go to [vercel.com/new](https://vercel.com/new)
2. Import **VVVULTURE/Red-Portal** from GitHub
3. Leave all settings at their defaults — Vercel will auto-detect the static site
4. Click **Deploy**

## Local Preview

No build step needed. Just open `index.html` in your browser, or serve it with any static file server:

```bash
npx serve .
```
